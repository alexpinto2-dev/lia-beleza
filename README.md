# Lia Beleza ✨

**A recepcionista IA mais simpática do Brasil**  
Agendamentos por WhatsApp 24h para salões de beleza.

Feito com carinho em Aracaju/SE ❤️

## Tecnologias
- Next.js 15 + TypeScript
- Supabase (Auth + Banco)
- Groq/Llama 3 (IA da Lia)
- Evolution API (WhatsApp)
- Tailwind + shadcn/ui

## Como rodar
1. `npm install`
2. Copie `.env.example` → `.env.local` e preencha
3. `npm run dev`

Pronto para lotar sua agenda!
